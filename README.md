Usage:
```
// Create a common (core) tile 3 cells width, and 2 cells in height
multiboard_common(3, 2);

// Create a side tile tile 3 cells width, and 2 cells in height. Tooth are on
// the right side
multiboard_side(3, 2);

// Create a corner 3 by 2 tile
multiboard_side(3, 2);
```
